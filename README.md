# GPIO Monitor

일렉트론 기반 멀티캐스트 GPIO 모니터링 애플리케이션

## 기능

- **멀티캐스트 디바이스 검색**: 239.255.0.1:5555로 멀티캐스트 요청을 보내 네트워크상의 GPIO 디바이스를 검색합니다.
- **실시간 디바이스 정보 표시**: 검색된 디바이스의 IP, MAC 주소, TCP 포트 정보를 실시간으로 표시합니다.
- **웹 브라우저 연동**: 디바이스 카드를 클릭하면 해당 디바이스의 웹 인터페이스를 기본 브라우저로 엽니다.

## 설치 방법

```bash
# 의존성 설치
npm install
```

## 실행 방법

```bash
# 애플리케이션 실행
npm start
```

## 사용 방법

1. 애플리케이션을 실행합니다.
2. **"GET - 디바이스 검색"** 버튼을 클릭합니다.
3. 네트워크상의 GPIO 디바이스가 자동으로 검색되어 화면에 표시됩니다.
4. 디바이스 카드를 클릭하면 웹 브라우저로 해당 디바이스에 접속할 수 있습니다.

## 프로토콜

### 멀티캐스트 설정

- **주소**: 239.255.0.1
- **포트**: 5555

### 메시지 형식

#### 요청 (디바이스 검색)

```
[0x01][0x00][0x00][0x00]
```

- 첫 바이트: DEVICE_INFO_REQUEST (0x01)

#### 응답 (디바이스 정보)

```
[msg_type][device_id][ip(4바이트)][mac(6바이트)][tcp_port(2바이트)]
```

- msg_type: DEVICE_INFO_RESPONSE (0x02)
- device_id: 디바이스 ID (1바이트)
- ip: IP 주소 (4바이트)
- mac: MAC 주소 (6바이트)
- tcp_port: TCP 포트 번호 (2바이트, big-endian)

## 디바이스 요구사항

이 애플리케이션은 [pico_gpio](https://github.com/jkjh8/pico_gpio) 장비와 호환되도록 설계되었습니다.
